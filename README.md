# Salary Prediction using Simple Linear Regression

## Overview

This project predicts employee salary based on years of experience using Simple Linear Regression.

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Workflow

- Load Dataset
- Data Exploration
- Train-Test Split
- Model Training
- Prediction
- Evaluation
- Visualization

## Result

R² Score ≈ 0.90
